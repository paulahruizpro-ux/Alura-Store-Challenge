# Introdução

O Alura Store Challenge foi um projeto desenvolvido dentro da Especialização de Data Science da Oracle Next Education, com o objetivo de aplicar os conhecimentos iniciais em Python, Pandas e Matplotlib. 

O seu objetivo foi apoiar a tomdade de decisão do Senhor João sobre qual das suas lojas fechar, tendo em vista iniciar um novo empreendimento.

# Etapas desenvolvidas

* Calcular o faturamento de cada loja;
* Identificar a quantidade de vendas por categoria de produto;
* Observar a média de avaliação de cada loja;
* Localizar e visualizar os produtos mais e menos vendidos em cada loja;
* Calcular o frete médio por loja.
* Com base nesses procedimentos e na análise dos achados, pretendeu-se fornecer uma recomendação que apoie a decisão do Senhor João sobre qual loja deve vender.

# Ferramentas usadas

* `Python`
* `Pandas`
* `Matplotlib`
* `Google Colaboratory`
